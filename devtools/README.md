# DevTools 开发技能

用于 [DevTools](https://github.com/axfinn/devtools) 项目的 Claude Code 开发技能。

## 功能

- 遵循 DDD 领域驱动设计
- 提供前后端开发模板
- 包含代码规范和最佳实践
- API 设计原则
- 添加新工具的完整流程

## 安装

```bash
# 复制到你的项目
mkdir -p .claude/skills
cp -r devtools .claude/skills/
```

## 使用

```
/dev 添加一个 Base58 编解码工具
```

## 适用项目

- Vue 3 + Vite + Element Plus 前端
- Go + Gin + SQLite 后端
- Docker 部署
