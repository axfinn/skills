# Blog Sync Skill

将开发信息同步到 Hugo 博客的技能。

## 功能

- 创建技术博客文章
- 更新动态 (Moments)
- 发布到 GitHub Pages

## 安装

```bash
claude mcp add-skill https://github.com/axfinn/skills/tree/main/blog-sync
```

## 使用

```
/blog 写一篇关于今天开发的技术文章
/blog 把这个功能的实现过程同步到博客
/blog 发布一条动态
```

## 目标博客

- 地址: https://blog.jaxiu.cn
- 引擎: Hugo
- 部署: GitHub Pages
